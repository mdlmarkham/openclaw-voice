🧠 Ask Métis

Hands-free voice chat with your AI agents via Siri.

Setup (30 seconds)

1. Open the Shortcuts app → tap +
2. Add action: Dictate Text
Language: English (US) · Stop listening: After Pause
3. Add action: Get Contents of URL
URL:
https://openclaw.tailf9480.ts.net/voice/api/chat
Method: POST · Body: JSON
Header: Content-Type: application/json
JSON body:
{"text": "Dictated Text", "agent": "metis"}
Tap text value → select Dictated Text variable
4. Add action: Play Audio
5. Name it Ask Metis
Settings → Siri → add phrase: "Ask Metis"

Use It

Say "Hey Siri, Ask Metis" → speak your question → hear the answer.

Other Agents

Change the agent value in the JSON body:

🧠 metis
🌍 atlas
🔧 hephaestus
📚 clio
📰 deepthought
💜 mara
TIP: Create one shortcut per agent with different Siri phrases: "Ask Metis", "Ask Atlas", etc.

API Reference

POST /api/chat — Full pipeline (LLM + TTS)
Returns: WAV audio
{"text": "your question", "agent": "metis"}
POST /api/speak — TTS only
Returns: WAV audio
{"text": "text to speak", "agent": "metis"}
